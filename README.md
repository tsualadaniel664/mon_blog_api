# API Blog Flask

Ce projet est une **API backend pour un blog simple**, développée avec **Python et Flask**, avec une interface web permettant de **créer, lire, modifier et supprimer des articles**.  

Chaque article contient :  
- `titre`  
- `contenu`  
- `auteur`  
- `date_creation`  
- `categorie`  
- `tags`

1) pour lancer l'application, ouvrir le navigateur et taper "python app.py"
2) Ouvrir le navigateur :
http://127.0.0.1:5000/articles-html
Actions disponibles :
Ajouter un article
Rechercher des articles par titre, auteur ou catégorie
Modifier un article
Supprimer un article
3) Fonctionnalités
CRUD complet pour les articles
Interface web avec tableau HTML
Recherche multi-critères (titre, auteur, catégorie)
Stockage dans SQLite
Formulaires de création et modification intégrés
